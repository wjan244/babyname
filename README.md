# Baby Names Data Visualization Project

This repository contains data visualizations developed as part of the Data Visualization course (P4) at **Télécom Paris**. It explores the evolution, regional distribution, and gender trends of baby names in France using **Altair**.

## Visualizations & Core Questions

The project addresses the following three exploratory visualization exercises:

### Visualization 1: Temporal Evolution
* **Questions:** How do baby names evolve over time? Are there names that have consistently remained popular or unpopular? Are there some that were suddenly or briefly popular/unpopular? Are there trends over time?
* **File:** `viz1.py`

### Visualization 2: Regional Effects
* **Questions:** Is there a regional effect in the data? Are some names more popular in specific regions or departments? Are popular names generally popular across the whole country?
* **File:** `viz2.py`

### Visualization 3: Gender Effects
* **Questions:** Are there gender effects in the data? Does the popularity of names given to both sexes evolve consistently? 
* **File:** `viz3.py`

## How to Run and View the Visualizations

The visualizations are pre-rendered and exported as interactive **HTML formats** embedded within the scripts or rendered locally. 

This project uses **`uv`** for fast and reliable Python dependency management.

### Prerequisites
Make sure you have `uv` installed. If not, you can install it via curl or Homebrew:
```bash
pip install uv  # or 'brew install uv' on macOS
